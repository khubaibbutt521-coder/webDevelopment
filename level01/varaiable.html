learning variables

variables types

  • VAR= Var is an old type .
  • Let= Let is a keyword in which we can make changes.
  • Const= Const is a keyword in which its value doesn't change and remain constant.

variables are used to store data in programming languages. they can hold different types of data,
such as numbers, strings, and booleans.variables are essential for writing programs that can perform
calculations, manipulate data, and make decisions based on user input.

in most programming languages, you can create a variable by assigning a value to it using the equals sign (=). for example:

            x = 10
            name = "Ali"
            is_student = True

in this example, we have created three variables: x, name, and is_student. x holds the integer value 10, name holds the string "Ali",
and is_student holds the boolean value True.

you can also perform operations on variables. for example:

            y = x + 5
            greeting = "Hello, " + name

in this example, we have created a new variable y that holds the result of adding 5 to x, which will be 15.
we also created a variablegreeting that concatenates the string "Hello, " with the value of name, resulting in "Hello, Ali".
